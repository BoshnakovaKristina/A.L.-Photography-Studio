# Photography Demo Website

Static illustrative photography website built with HTML, CSS and JavaScript.

This is a fictional demo concept for portfolio presentation. All names, contact details, quotes and stories are fictional.

## Files
- `index.html` — home page
- `portfolio.html` — portfolio gallery with filters
- `services.html` — services page
- `about.html` — about page
- `journal.html` — journal overview
- `lakecomo.html`, `paris.html`, `amalfi.html` — demo story pages
- `contact.html` — contact page
- `style.css` — design and responsive styles
- `script.js` — menu, loader, filters and form demo interactions
- `assets/` and `img/` — demo images and SVG assets

## Usage
Open `index.html` in a browser or upload the full folder to hosting.
